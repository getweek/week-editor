# Week Editor

## Plugins

* **BasicMarksPlugin**  
Adds support for bold, italic and underlined text.

* **DnDPlugin**  
Allows to reorder blocks.

* **FloatingUiPlugin**  
Allows to setup a floating menu with quick actions.

* **LinksPlugin**  
Adds support for links.

* **NodeIdPlugin**  
Assigns each block an unique identifier.

* **SoftBreakPlugin**  
Allows to insert soft break.
